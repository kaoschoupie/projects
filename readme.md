This is my repository for my small projects

Cybertruck blender is a simple Blender render of a Cybertruck running on OpenGL</br>
Written in C++

filecypher is a simple bytewise cypher/decypher for any file, the encryption is a polyalphabetic based on a string</br>
Written in C

reuters-21578_indexer is a program that creates an inverted index of the Reuters-21578 corpus</br>
Written in Python

stockscraper is a simple webscrape of finance.yahoo.com/quote/ to get the quote of any stock and log it with the time it was scraped</br>
Written in java

txtcypher is a simple cypher/decypher program for txt file contents, the encryption is polyalphabetic based on an int</br>
Written in C

warzone is a risk-like command-line game done in the context of a school project which explains some of the weirder design choices</br>
Written in C++
